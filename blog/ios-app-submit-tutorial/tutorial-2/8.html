Deployment Target
(4) to 9.0 at the top:
Note that deployment target is a fancy way of saying “the minimum version of
iOS that your code supports”. Be careful though, because if, for example, you
set this to iOS 8.0 but use an API that is only available on iOS 9 without
checking first, your app will crash! The safest thing to do is to test your code on
a device running the oldest version of iOS you want to support.
