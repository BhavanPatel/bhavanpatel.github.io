Apple recommends that you use “a reverse-domain name style string” for
the Bundle ID. For an Explicit App ID, the suggested format is
“com.domainname.appname”; for a Wildcard App ID, the suggested format
is “com.domainname.*”.
•
Remember, if you use a Wildcard App ID, you won’t be able to use any of
the neat services normally available, such as push notifications or in-app
purchases. You might not plan to use these services now, but if you change
your mind, you won’t be able to change your App ID without creating a new
app.
